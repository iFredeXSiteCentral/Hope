# **App Name**: Hope's RecipeShare

## Core Features:

- Homepage: Landing page showcasing featured recipes.
- Recipe Upload Form: Form to upload recipe name, ingredients, instructions. All fields required.
- Recipe Display: Page to display individual recipes including images and full instructions.
- Personalized Recipe Recommendations: AI Recipe Suggestion Tool: Provide personalized recipe recommendations based on user preferences, dietary restrictions, and available ingredients.
- Footer: Displays "Hope's RecipeShare Copyright 2025. Powered by iFredeX Creative Hub" on all pages.

## Style Guidelines:

- Primary color: Warm beige (#F5F5DC) for a comforting, home-style feel.
- Secondary colors: Soft greens (#E8F5E9) and light browns (#D7CCC8).
- Accent: Terracotta (#E07A5F) for calls to action and highlights.
- Clean and readable font for recipe text and descriptions.
- Use simple, outline-style icons related to cooking and ingredients.
- Clean and organized layout with clear sections for recipe details and instructions.
- Subtle transitions and animations for a smooth user experience.
